Em Busca Da Web (Uma historia de amor e pitufos)

Era uma vez em tão tão distante havia um pitu... desculpe... havia uma tribu de pitufos... que sonhavam em ser programadores, Vulgo Devs Web.

Um belo dia andando pelo bosque da cidade pitalandia, eles encontraram uma cobra chamada python que para os intimos era chamado Java e o pitufinho disse NÂO eu vou experemintar o css

Mas logo ele mudou de ideia e acabou comendo o fruto proibido e acabou conhecendo o capitão Java

Após ele conhecer o capitão Java ele decidiu programar por 40 dias e 40 noites, ele foi foi expulso do VsCode pelo HTML, quando ele achou que a tempestade havia
acabado veio um grande diluvio chamado git e devastou todo o seu conhecimento pela parede e veio a chuva forte e o derrubou... Tragico FIM para pitufinho
